INSTALLATION GUIDE:- 
1) Right click on Activate file and click on edit.
2) Replace ENTER_PATH_HERE with the path where you extracted the zip file and then save the file.
3) Close Activate.cmd double click to run it.
4) Enter the path of folder where you extracted zip file and then apply.
PROGRAM WILL BE INSTALLED SUCCESSFULLY.

THERE ARE TWO VERSION'S OF APPLICATION,
1. UV_Assistant
2. UV_Assistant_Turbo

*double click on 'UV_Assistant launcher' or 'UV_Assistant_Turbo launcher' to launch the application. You can also create shortcuts of these applications on your desktop.
